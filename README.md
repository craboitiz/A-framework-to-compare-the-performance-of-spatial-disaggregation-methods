# A framework to compare the performance of spatial disaggregation methods of statistical climate downscaling in South America

This repository contains the code and supporting information associated with the manuscript:

**A framework to compare the performance of spatial disaggregation methods of statistical climate downscaling in South America** 

**Authors:** Cristóbal Aboitiz, Cristián Chadwick, Jonathan Barichivich, Jorge Perez-Quezada

**Status:** Supporting information for manuscript submission

## Repository structure

- `code/`: Scripts used for data preprocessing, spatial disaggregation, evaluation metrics, and figure/table generation.
- `data_description/`: Description of input datasets, file structure, and data availability.
- `supporting_information/`: Supplementary figures, tables, and methodological details.

## Reproducibility

The scripts and supporting files are provided as a reference implementation of the workflow used in the study. File paths, input datasets, spatial and temporal domains, resolution scales, and disaggregation methods should be modified as needed to match the specific case study, datasets, and analysis objectives of the user.

The order in which the scripts should be run depends on the selected configuration and on the available input files. Therefore, users should review each script and adjust the corresponding parameters before execution.

## Data availability

Large input datasets are not stored in this repository. The original datasets used in this study must be downloaded from their official sources.
