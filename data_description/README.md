# Data description

This folder describes the input datasets, file structure, and data availability associated with the study:

**A framework to compare the performance of spatial disaggregation methods of statistical climate downscaling in South America**

Large raw datasets are not stored in this repository due to file size and/or redistribution restrictions. Users should obtain the original datasets from their official sources before running the scripts.

## Data sources

The study may require datasets such as:

- Gridded observational datasets
- Reanalysis products
- Global climate model outputs
- Processed climate variables used for spatial disaggregation and bias correction

Users should verify the availability, version, spatial resolution, temporal resolution, and usage license of each dataset.

## Expected information

This folder can include documentation about:

- Dataset names and versions
- Download sources
- Spatial and temporal coverage
- Variable names and units
- File naming conventions
- Preprocessing steps

## Notes

The exact input files, spatial domain, temporal period, and variables should be modified according to the specific case study and analysis objectives of the user.
